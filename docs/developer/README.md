## Current Baseline

The repository is still in foundation setup. The only required local validation commands today are:

```powershell
python tools/validate_foundation.py
python contracts/validate_contracts.py
```

Backend, admin frontend, mobile, Rust, and deployment commands should be added here when those scaffolds are implemented.
