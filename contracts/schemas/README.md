# JSON Schemas

Add versioned JSON Schemas here. Start with module manifest, theme, form, screen, app, workflow, sync rule, and deployment package schemas.

Initial schemas should use the conventions in `../SCHEMA_CONVENTIONS.md`.

Use `v1/` for the first schema version:

```text
contracts/schemas/v1/
```

Do not add unversioned schema files.
